## 📝 Sobre o Projeto

Este é um projeto que desenvolvi para aplicar meus estudos iniciais de **Python** voltados para a área de **Cibersegurança**. O objetivo foi criar uma ferramenta simples, mas que seguisse boas práticas de segurança digital e criptografia.

### 🚀 O que este script faz?
O script gera senhas aleatórias de alta complexidade, combinando letras (maiúsculas e minúsculas), números e caracteres especiais. 

### 🔐 Por que foquei em Cibersegurança?
Diferente de geradores comuns que usam a biblioteca `random`, eu utilizei o módulo **`secrets`**. Escolhi essa biblioteca porque ela é projetada para ser **criptograficamente segura**, o que significa que a senha gerada é muito mais difícil de ser prevista ou quebrada por ataques de força bruta.
